# Project 3 — TMDB 电影数据分析

TMDB Top 300 电影数据统计分析与可视化。

## 功能

- 电影数量按年份/语言/类型分布
- 评分分布分析
- 4 面板可视化（折线 + 柱状 + 饼图）
- Jupyter Notebook 交互式分析

## 技术栈

- **数据**: Pandas · **可视化**: Matplotlib · **环境**: Jupyter

## 运行

```bash
jupyter notebook project_03_data_analysis/movie_analysis.ipynb
python project_03_data_analysis/movie_charts.py
```
