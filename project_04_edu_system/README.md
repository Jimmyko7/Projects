# Project 4 — 教务管理系统

纯 OOP（面向对象）实现的控制台教务管理系统。

## 功能

- 学生成绩增删改查
- 按姓名搜索学生
- 成绩校验（0~100 范围限制）
- 控制台菜单交互

## 技术栈

- **语言**: Python（纯 OOP，无第三方依赖）

## 运行

```bash
python project_04_edu_system/main.py
```

## 架构

```
StuGrade (数据类)          EduManagement (业务类)
├── 姓名/成绩              ├── 添加/删除/修改/查询
└── 成绩校验                └── 菜单交互
```
